# safari-highlighter

Lightweight keyboard-driven highlighter for macOS Safari

### Installation
1. Install the [Userscripts](https://itunes.apple.com/app/id1463298887) Safari extension.
2. Open the **Userscripts Directory** from the Safari toolbar.
3. Move `safari-highlighter.user.js` into that folder.

### Shortcuts
* **H** — Highlight selection (Hold to cycle colors)
* **R** — Remove last highlight
* **C** — Copy all highlights to clipboard
